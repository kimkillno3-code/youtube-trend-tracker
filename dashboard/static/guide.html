시작하기

준비물

항목설명
Python3.9 이상
YouTube API 키Google Cloud Console에서 발급
YouTube Data API v3Google Cloud Console → API 및 서비스 → 라이브러리에서 활성화

API 키 등록하기

프로젝트 폴더에 .env 파일을 만들고 아래처럼 입력하세요.

YOUTUBE_API_KEY=발급받은_API_키
확인 방법 — 설정 페이지에서 API 키가 등록됐는지 바로 확인할 수 있어요 (마스킹 처리돼서 안전해요)

실행하기

pip install -r requirements.txt
streamlit run dashboard/app.py

브라우저에서 http://localhost:8501로 접속하면 바로 쓸 수 있어요.

트렌드 대시보드

실시간 트렌드

한국 YouTube에서 지금 인기 급상승 중인 영상을 가져와요.

  1. 실시간 인기 영상 가져오기 버튼 클릭
  2. 핵심 지표 확인 — 영상 수, 평균·최고 조회수, 인기 카테고리
  3. 카테고리 필터로 관심 분야만 골라보기
  4. 영상 카드 클릭하면 YouTube로 바로 이동
  5. 엑셀 다운로드로 데이터 저장
API ~2 유닛 · 하루 5,000회 이상 가능

주간 인기

인기 급상승 영상 중 조회수 TOP 30을 한눈에 비교할 수 있어요.

  • 실시간 중복 제외 토글을 켜면 새로운 영상만 볼 수 있어요
API ~2 유닛

분석 리포트

수집된 데이터를 기반으로 트렌드 인사이트를 제공해요.

  • 카테고리별 분포 (도넛 차트)
  • 자주 쓰인 태그 TOP 15
  • 조회수 분포 · 참여율 TOP 10
  • 채널별 등장 횟수 · 영상 길이 분포
  • AI 인사이트 — 평균 영상 길이, 소규모 채널 성과
API 사용량 0 · 무제한 분석
4

키워드 검색

경쟁 영상 찾기

  1. 검색어 입력 — 예: AI, 먹방, 브이로그
  2. 기간 선택 — 최근 1주일 / 1개월 / 3개월 / 전체
  3. 영상 길이 선택 — 전체 / 4분 미만 / 4~20분 / 20분 이상
  4. 검색하기 클릭

결과에서 할 수 있는 것

  • 핵심 지표 — 평균 조회수, 최고 조회수, 평균 길이, 소채널 성과
  • 인사이트 — 인기 카테고리, 많이 쓰인 태그, 소채널 분석
  • 정렬 — 조회수순 / 최신순 / 관련도순
  • 카테고리 필터 — 원하는 카테고리만 골라보기
  • 엑셀 다운로드 / 썸네일 모아받기
API ~102 유닛 · 하루 약 98회 검색 가능
참고 — 키워드 검색은 search.list API를 사용해서 1회당 100유닛이 필요해요. 가장 비용이 높은 기능이니 신중하게 사용하세요.
5

콘텐츠 아이디어

급상승 키워드

Google Trends 데이터를 기반으로 지금 급상승 중인 키워드를 보여줘요. 남들보다 빠르게 트렌드를 잡을 수 있어요.

API 사용량 0

키워드 갭 분석

검색 수요는 높은데 영상 공급이 부족한 키워드를 찾아줘요. 점수가 높을수록 블루오션이에요.

활용 팁 — 갭 점수가 높은 키워드로 영상을 만들면, 경쟁은 적고 수요는 높아서 조회수를 빠르게 끌어올릴 수 있어요.

시즌 캘린더

시기별로 반복되는 인기 주제를 미리 파악할 수 있어요. 콘텐츠 일정을 앞서서 준비하세요.

6

설정

  • API 키 상태 — 등록된 키를 마스킹해서 안전하게 표시 (앞 4자리 + *** + 뒤 4자리)
  • 일일 할당량 — 오늘 사용량, 잔여량, 사용률을 프로그레스 바로 확인
  • 최근 수집 기록 — 최근 10건의 수집 이력 (유형, 시간, 영상 수, 사용량, 상태)
할당량 초기화 — 매일 태평양 시간 자정(한국 시간 오후 4~5시)에 자동으로 리셋돼요.
7

성과 별점

영상 카드의 별점은 구독자 대비 조회수 비율로 매겨요.

별점조회수 / 구독자의미
★★★★★50배 이상초대박
★★★★10배 이상대박
★★★2배 이상우수
★★0.5배 이상보통
★그 외기본
블루오션 발견법 — 구독자 5만 이하 소규모 채널이 ★★★★ 이상을 받았다면, 해당 주제가 블루오션일 가능성이 높아요.

API 할당량 관리

기능별 API 비용

기능유닛하루 가능 횟수
실시간 인기 영상 가져오기~2~5,000회
주간 인기 영상 분석~2~5,000회
키워드 검색~102~98회

API 없이 쓸 수 있는 기능

아래 기능들은 이미 수집된 데이터를 사용해서 할당량 소모가 전혀 없어요.

  • 분석 리포트 (차트, 인사이트)
  • 카테고리 필터링 · 정렬 변경
  • 엑셀 다운로드 · 썸네일 모아받기
  • 콘텐츠 아이디어 (급상승 키워드, 갭 분석, 시즌 캘린더)
  • 설정 페이지 전체 · 라이트/다크 모드 전환

할당량 아끼는 법

  1. 검색은 신중하게 — 1회에 ~102유닛이에요. 하루 98회까지 가능
  2. 트렌드 수집은 부담 없이 — 1회에 ~2유닛으로 아주 저렴해요
  3. 분석 리포트 적극 활용 — 한 번 수집한 데이터는 무제한 분석 가능
  4. 설정에서 사용량 확인 — 70% 이상이면 안내 메시지가 떠요

문제 해결

이런 메시지가 보이면이렇게 해결하세요
API 키가 설정되지 않았어요 프로젝트 루트에 .env 파일을 만들고 YOUTUBE_API_KEY=키값 입력
일일 할당량을 초과했어요 한국 시간 오후 4~5시 이후 자동으로 초기화돼요. 그때까지 분석 리포트를 활용하세요
API 키 권한이 부족해요 Google Cloud Console에서 YouTube Data API v3이 활성화됐는지 확인
네트워크 연결을 확인해주세요 인터넷 연결 상태를 확인한 뒤 다시 시도
