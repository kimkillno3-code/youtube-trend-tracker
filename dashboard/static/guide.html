트렌드 대시보드

실시간 트렌드

한국 YouTube에서 지금 인기 급상승 중인 영상을 가져와요.

  1. 실시간 인기 영상 가져오기 버튼 클릭
  2. 핵심 지표 확인 — 영상 수, 평균·최고 조회수, 인기 카테고리
  3. 카테고리 필터로 관심 분야만 골라보기
  4. 영상 카드 클릭하면 YouTube로 바로 이동
  5. 엑셀 다운로드로 데이터 저장
API ~2 유닛 · 하루 5,000회 이상 가능

주간 인기

인기 급상승 영상 중 조회수 TOP 30을 한눈에 비교할 수 있어요.

  • 실시간 중복 제외 토글을 켜면 새로운 영상만 볼 수 있어요
API ~2 유닛

분석 리포트

수집된 데이터를 기반으로 트렌드 인사이트를 제공해요.

  • 카테고리별 분포 (도넛 차트)
  • 자주 쓰인 태그 TOP 15
  • 조회수 분포 · 참여율 TOP 10
  • 채널별 등장 횟수 · 영상 길이 분포
  • AI 인사이트 — 평균 영상 길이, 소규모 채널 성과
API 사용량 0 · 무제한 분석
2

키워드 검색

경쟁 영상 찾기

  1. 검색어 입력 — 예: AI, 먹방, 브이로그
  2. 기간 선택 — 최근 1주일 / 1개월 / 3개월 / 전체
  3. 영상 길이 선택 — 전체 / 4분 미만 / 4~20분 / 20분 이상
  4. 검색하기 클릭

결과에서 할 수 있는 것

  • 핵심 지표 — 평균 조회수, 최고 조회수, 평균 길이, 소채널 성과
  • 인사이트 — 인기 카테고리, 많이 쓰인 태그, 소채널 분석
  • 정렬 — 조회수순 / 최신순 / 관련도순
  • 카테고리 필터 — 원하는 카테고리만 골라보기
  • 엑셀 다운로드 / 썸네일 모아받기
API ~102 유닛 · 하루 약 98회 검색 가능
참고 — 키워드 검색은 1회당 100유닛이 필요해요. 가장 비용이 높은 기능이니 신중하게 사용하세요.
3

콘텐츠 아이디어

급상승 키워드

Google Trends 데이터를 기반으로 지금 급상승 중인 키워드를 보여줘요. 남들보다 빠르게 트렌드를 잡을 수 있어요.

API 사용량 0

키워드 갭 분석

검색 수요는 높은데 영상 공급이 부족한 키워드를 찾아줘요. 점수가 높을수록 블루오션이에요.

활용 팁 — 갭 점수가 높은 키워드로 영상을 만들면, 경쟁은 적고 수요는 높아서 조회수를 빠르게 끌어올릴 수 있어요.

시즌 캘린더

시기별로 반복되는 인기 주제를 미리 파악할 수 있어요. 콘텐츠 일정을 앞서서 준비하세요.

4

설정

  • API 키 상태 — 등록된 키를 마스킹해서 안전하게 표시
  • 일일 할당량 — 오늘 사용량, 잔여량, 사용률을 프로그레스 바로 확인
  • 최근 수집 기록 — 최근 10건의 수집 이력 확인
할당량 초기화 — 매일 한국 시간 오후 4~5시에 자동으로 리셋돼요.
